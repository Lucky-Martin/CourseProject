The project includes the following features:

Quaters
- Able to select crew members
- Move selected members to simulator
- Move selected members to mission control

Simulator
- Select members
- Train memebers for additional experience
- Move members to quaters

Mission control
- Select members to participate
- Return memebers to quaters
- Launch mission

Recruit crew members
- Input crew name
- Choose crew specialization

Statistics
- Stats for missions, crew etc.

The project was created using Android Studio and Java. Also we have used AI for help in certain cases, but majority is written by us.
